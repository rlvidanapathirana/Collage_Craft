# ✦ CollageCraft – Premium Free Collage Maker

A beautiful, premium, and fully responsive web-based collage maker built entirely in a single HTML file using Vanilla JavaScript and CSS. No dependencies, no watermarks, and no servers required—everything runs locally in your browser.

## ✨ Features

* **Custom Grid Builder:** Dynamically adjust rows and columns to create your perfect layout.
* **Drag & Pan Positioning:** Easily click and drag uploaded photos to position them perfectly within their slots.
* **Advanced Photo Adjustments:** Fine-tune your images with built-in sliders for Zoom, Brightness, Contrast, and Saturation.
* **One-Click Color Grades:** Apply stunning global presets (Warm, Cool, Noir, Cinematic, Mint, etc.) instantly.
* **Adjustable Spacing:** Control the gap between images with pixel-perfect precision.
* **Social Media Ready Sizes:** Pre-configured canvas sizes for Instagram Square, Facebook Cover, 16:9, Portrait, Story, and 4×6 Prints.
* **High-Resolution Export:** Download your masterpiece in high-quality **PNG** (lossless) or **JPEG** (optimized) formats, strictly without watermarks.
* **Glassmorphism UI:** A sleek, modern, and premium user interface featuring backdrop filters and smooth CSS transitions.
* **100% Mobile Responsive:** Carefully optimized for touch devices with a scrollable toolbar and intuitive touch-and-drag mechanics.
